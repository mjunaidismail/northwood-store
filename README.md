# NORTHWOOD

A modern and minimalist ecommerce landing page built using pure HTML and CSS.

This project was created to practice clean UI design, layout structuring, hover animations, spacing systems, and modern frontend styling without using any framework.

The overall goal was simple:
build something that feels premium, clean, and realistic — like an actual lifestyle/product brand website.

---

## Preview

Northwood focuses on:
- minimal design
- clean typography
- soft neutral colors
- modern product presentation
- subtle animations

---

## Features

- Modern ecommerce-style landing page
- Clean navigation bar
- Hero section with CTA
- Feature/service highlights
- Product showcase cards
- Smooth hover animations
- Minimal and premium UI styling
- Flexbox based layout system

---

## Tech Stack

- HTML5
- CSS3
- Flexbox

No frameworks or libraries were used for styling.

---

## What I Practiced In This Project

While building this project, I mainly focused on:

- writing cleaner HTML structure
- improving CSS organization
- creating reusable styling patterns
- adding subtle professional animations
- understanding spacing and visual hierarchy
- building UI that feels realistic instead of tutorial-like

Thoda minimal rakhne ki try ki taake UI clean aur premium feel de.
